# Quiz Game Project

## 1. 프로젝트 개요
이 프로젝트는 Python으로 만든 콘솔 기반 퀴즈 게임입니다.  
사용자는 메뉴를 통해 퀴즈를 풀고, 새로운 퀴즈를 추가하고, 등록된 퀴즈 목록과 최고 점수를 확인할 수 있습니다.  
또한 프로그램의 상태를 `state.json` 파일에 저장하여, 프로그램을 종료한 뒤 다시 실행해도 퀴즈 데이터와 최고 점수가 유지되도록 구현했습니다.

## 2. 퀴즈 주제 선정 이유
퀴즈 주제로 야구를 선택했습니다.  
야구는 기본 규칙이 비교적 명확하고 객관식 문제로 만들기 쉬워서 퀴즈 형식에 잘 어울린다고 생각했습니다.  
또한 문제, 선택지, 정답을 구조적으로 정리하기에 적합하여 프로그램 설계 연습에도 도움이 된다고 판단했습니다.

## 3. 실행 방법

### 실행 환경
- Python 3.10 이상

### 실행 명령어
```bash
python main.py
```

## 4. 기능 목록
- 퀴즈 풀기
  - 등록된 퀴즈를 순서대로 풀 수 있습니다.
  - 정답/오답 여부와 최종 점수를 확인할 수 있습니다.

- 퀴즈 추가
  - 문제, 선택지 4개, 정답 번호를 입력해 새 퀴즈를 추가할 수 있습니다.

- 퀴즈 목록 확인
  - 현재 저장된 퀴즈 목록을 확인할 수 있습니다.

- 점수 확인
  - 현재까지의 최고 점수를 확인할 수 있습니다.

- 데이터 저장/불러오기
  - 퀴즈 데이터와 최고 점수를 `state.json`에 저장하고 다시 불러옵니다.

- 예외 처리
  - 잘못된 숫자 입력, 빈 입력, 범위 밖 입력을 처리합니다.
  - `Ctrl + C`, `EOFError` 발생 시 가능한 범위에서 저장 후 안전하게 종료합니다.

## 6. 데이터 파일 설명
이 프로젝트는 프로그램 상태를 프로젝트 루트의 `state.json` 파일에 저장합니다.

### 파일 경로
- `state.json`

### 역할
- 등록된 퀴즈 목록 저장
- 최고 점수 저장

### 데이터 구조 예시
```json
{
    "quizzes": [
        {
            "question": "야구에서 한 팀이 공격할 때 아웃 3개를 당하면 어떻게 될까?",
            "choices": [
                "경기가 종료된다",
                "수비를 계속한다",
                "공수 교대가 된다",
                "점수가 1점 추가된다"
            ],
            "answer": 3
        }
    ],
    "best_score": 5
}
```

## 7. 클래스 설명

### Quiz
개별 퀴즈 1개를 표현하는 클래스입니다.

- `question`: 문제
- `choices`: 선택지 목록
- `answer`: 정답 번호

주요 메서드:
- `display()`
- `is_correct()`
- `to_dict()`
- `from_dict()`

### QuizGame
게임 전체를 관리하는 클래스입니다.

주요 기능:
- 메뉴 출력
- 사용자 입력 처리
- 퀴즈 진행
- 퀴즈 추가
- 퀴즈 목록 출력
- 최고 점수 확인
- 파일 저장 및 불러오기
